Política de Privacidade

Última atualização: março de 2025

1. Sobre esta política

Esta Política de Privacidade descreve como o PilotoFinanceiro trata os seus dados ao usar o aplicativo.

2. Dados que coletamos

O PilotoFinanceiro não coleta, não armazena e não transmite nenhum dado pessoal para servidores externos. O aplicativo funciona 100% offline.

Todos os dados inseridos no app ficam armazenados exclusivamente no armazenamento local do seu dispositivo.

3. Dados que NÃO coletamos

  • Nome, e-mail ou qualquer dado de identificação pessoal
  • Localização geográfica
  • Dados de uso ou comportamento dentro do app
  • Informações financeiras ou bancárias
  • Dados do dispositivo

4. Armazenamento local

Os dados ficam salvos localmente no armazenamento interno do seu celular (localStorage). O PilotoFinanceiro não tem acesso a esses dados — eles pertencem somente a você.

5. Acesso à internet

O PilotoFinanceiro não requer conexão com a internet para funcionar. A única situação em que a internet é usada é durante o download inicial do aplicativo.

A landing page utiliza o Google Analytics para medir o número de visitas e cliques conforme a Política de Privacidade do Google.

6. Compartilhamento de dados

Não compartilhamos nenhum dado com terceiros, pois não coletamos nenhum dado.

7. Seus direitos

Para apagar seus dados do app, basta desinstalar o aplicativo do seu dispositivo.
